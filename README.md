# PGA Leaderboard by Hai Nguyen

## Requirements

    Node v10.12.0
    NPM 6.4.1
    Mongo v3.6.3

## Installation

    npm install
    
## Development

### For API development:

    npm run api

Browse to http://localhost:5001/graphql

### For UI development:

    npm start

Browse to http://localhost:5000

### For test-driven development:

    npm run test
